1、只允许线性方程组
2、只支持'纯英文','英文_英文数字混合下标','纯英文\_'的变量形式形如：

- 纯英文：F $F$;Power $Power$
- 英文带下标（\_不可少）：F_1 $F_1$; F_a $F_a$; F_a1 $F_{a1}$

- 纯英文只带下划线：F_ 仅用于区分F与F_，类似与F与F‘这样的区分效果

3、不支持的部分情况

- 前缀是英文+数字混合：a1,b1，应该写成a_1,b_1
- 中文不允许出现在变量里

4、运算符

```
+ - * / ** ( )
```

** 是幂次

```
3**2 = 9
```

4、注意

- 变量名是区分大小写的：A_1 与a_1是不一样的
- 要保证方程的数量和变量的数量要相同
- 只提供数值解，不提供解析解

5、应用场景

- 受力平衡方程
- 电路KCL、KVL
- 线性代数作业
