1、只允许线性方程组
2、支持的变量名如下：

- 纯英文：F $F$;Power $Power$
- 英文带下标（\_不可少）：F_1 $F_1$; F_a $F_a$; F_a1 $F_{a1}$
- 纯英文只带下划线：F_ 仅用于区分F与F_，类似与F与F‘这样的区分效果
- 前缀是英文+数字混合：a1,b1

3、不支持的变量命名（部分情况）

- 以数字开头：3a
- 中文不允许出现在变量里

4、支持的运算符（加减乘除幂，括号运算）

```
+ - * / ** ( )
```
注意：*号不能省略
** 是幂次

```
3**2 = 9，9**(0.5)=3，注意：(-2)**(0.5)无解
```

4、注意

- 变量名是区分大小写的：A_1 与a_1是不一样的

- 要保证方程的数量要大于或者等于变量的数量，不然会报错

- 只提供数值解，不提供解析解，如果没有解会返回错误：

​	矩阵奇异（行列式为0），无法求解

5、应用场景

- 受力平衡方程
- 电路KCL、KVL
- 线性代数作业
